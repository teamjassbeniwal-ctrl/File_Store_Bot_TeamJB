# Contributing to FILE-SHARING-BOT

Thank you for your interest in contributing to FILE-SHARING-BOT 🎉  
We welcome all kinds of contributions and appreciate your support in improving this project.

Whether you are a beginner or an experienced developer, your help is valuable.

---

## 🚀 Ways to Contribute

You can contribute in many ways, including:

- Reporting bugs  
- Suggesting new features  
- Fixing issues  
- Improving documentation  
- Sharing ideas and feedback  

---

## 🧑‍💻 Development Process

We use GitHub to manage the project, track issues, and review contributions.

Follow these simple steps to contribute:

1. Fork the repository  
2. Create a new branch from `MAIN`  
3. Make your changes  
4. Test your code properly  
5. Commit your changes with a clear message  
6. Submit a Pull Request to the `develop` branch  

---

## ✅ Contribution Guidelines

- Make sure your code is clean and readable  
- Follow the existing project structure  
- Test your changes before submitting  
- Avoid unnecessary changes in unrelated files  
- Add comments where needed

## 🐞 Reporting Issues

We use GitHub Issues to track bugs and feature requests.

Open an issue here:  
https://github.com/TEAM-JB-official/File-Sharing-Bot/issues  

Please include:

- Clear description of the problem  
- Steps to reproduce  
- Screenshots (if possible)
